El login es el proceso mediante el cual un usuario se identifica en una aplicación utilizando sus credenciales. 
Esto permite verificar su identidad antes de acceder a funciones específicas.
 Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes en el sistema. 
 Solo los usuarios autorizados deben tener acceso a estas funciones.
 No se deben exponer públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
